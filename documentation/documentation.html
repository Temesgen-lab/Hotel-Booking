Overview

Template Name: Ashie Hotel

Purpose: A fully responsive hotel template designed to manage booking systems and provide a seamless experience for both hotel staff and customers.

Key Features:

  • Fully Responsive Layout
  • Reusable Components
  • Customizable Themes
  • Booking Form Integration
  • UI Elements
  • Clean and Organized Code

Installation Guide

Follow these steps to install and run Ashie Hotel template locally:

  • Step 1: Download and extract the template ZIP file.
  • Step 2: Open the project in your code editor.
  • Step 3: Install dependencies using npm install.
  • Step 4: Run the template using npm run dev.

Customization Guide

To customize the Ashie Hotel template:

  • Favicon: Replace the file at /public/hotel.ico.
  • Logo: Replace the image at /src/components/header/img/hotel-logo.png.
  • Color Scheme & Fonts: Modify values in /src/index.css.
